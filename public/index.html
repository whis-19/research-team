Research Team

Advancing Software Security & AI Research

Research Focus: Secure Code Generation

Our research team is pioneering innovative approaches to automated secure code generation through advanced AI-driven methodologies. We leverage state-of-the-art techniques, including Large Language Models (LLMs) and Retrieval-Augmented Generation (RAG), to automatically generate secure, vulnerability-free code that adheres to software security best practices. Our work combines cutting-edge software engineering principles with machine learning to enhance software security from the ground up, ensuring that generated code is resilient against common security threats. The team focuses on developing intelligent code generation systems that can detect and prevent Common Weakness Enumerations (CWEs) during the code creation process, providing context-aware security analysis and producing production-ready, secure code across multiple programming languages and frameworks.
